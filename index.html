Bot Documentation

Use of bot

/play - изменяет текущий баланс игрока.

/help - список команд.

/statistic - статистика игрока.

upgrade - прокачка игрока.

/upgradeinfo - информация о каждом аспекте прокачки.

/events - текущие события.

/top - глобальный топ

/localtop - локальный топ.

/batlez - сражение с другими игроками
